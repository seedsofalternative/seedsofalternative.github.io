# Seeds of Alternative

> Spreading alternative ideas, one seed at a time.

[https://seedsofalternative.com](https://seedsofalternative.com) — a personal blog covering Buddhism, Psychology, spirituality, sustainability, and conscious living.

## Tech stack

- [Jekyll](https://jekyllrb.com/) static site generator
- [GitHub Pages](https://pages.github.com/) for hosting
- Theme: [Mediumish](https://github.com/wowthemesnet/mediumish-theme-jekyll) ([docs](https://bootstrapstarter.com/template-mediumish-bootstrap-jekyll/))
- Key plugins: `jekyll-feed`, `jekyll-sitemap`, `jekyll-paginate`, `jekyll-seo-tag`, `jekyll-archives`

## Content

18 posts across two main categories:

| Category   | Topics                                                              |
|------------|---------------------------------------------------------------------|
| Buddhism   | Buddhist teachers & traditions, meditation, four noble truths, delusions, suffering |
| Psychology | Happiness, dopamine, introspection, nonviolent communication, emotions, values (5-part series) |

## Features

- **Search** — client-side full-text search via [Lunr.js](https://lunrjs.com/)
- **Comments** — Disqus integration on posts
- **Analytics** — Umami (privacy-friendly, self-hosted)
- **RSS feed** — `/feed.xml` (10 most recent posts)
- **Sitemap** — auto-generated by `jekyll-sitemap`
- **Star ratings** — optional per-post rating display
- **Table of contents** — auto-generated from headings via `_includes/toc.html`
- **YouTube embeds** — `{% include youtubePlayer.html id="..." %}`
- **Lazy image loading** — enabled globally

## Local development

### With Bundler

```bash
git clone git@github.com:seedsofalternative/seedsofalternative.github.io.git
cd seedsofalternative.github.io/
gem install bundler
bundle install
bundle exec jekyll serve --watch
```

### With Docker

```bash
docker-compose up
```

Site available at `http://localhost:4000`.

## Project structure

```
.
├── _posts/          # Blog posts (YYYY-MM-DD-title.md)
├── _pages/          # Static pages (about, categories, location pages, etc.)
├── _layouts/        # Page templates (default, post, page, archive)
├── _includes/       # Reusable components (postbox, share, toc, search, etc.)
├── _sass/           # SCSS partials
├── assets/
│   ├── css/         # Compiled styles + Bootstrap theme
│   ├── js/          # jQuery, Lunr, lazyload, theme JS
│   ├── fonts/       # Casper icon font
│   └── images/      # Logo, avatar, post images (organised by topic)
├── _config.yml      # Site configuration
├── Gemfile          # Ruby dependencies
├── docker-compose.yml
└── CNAME            # Custom domain: seedsofalternative.com
```

## Adding a post

Create `_posts/YYYY-MM-DD-your-title.md` with this front matter:

```yaml
---
layout: post
title: "Your Post Title"
author: pierre
categories: [Buddhism]   # or [Psychology], etc.
tags: [tag1, tag2]
image: assets/images/your-topic/cover.jpg
rating: 4               # optional, 0–5
---

Post content here...
```

## Deployment

Push to `main` — GitHub Pages builds and deploys automatically. The custom domain is configured via the `CNAME` file (`seedsofalternative.com`).
